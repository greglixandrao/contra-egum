# Contador Regressivo - Contra-Egun

Contador regressivo simples para a data de retirada do contra-egum.

## Data Final

## Tecnologias
- HTML5
- CSS3
- JavaScript Vanilla

Abra o arquivo `index.html` em seu navegador ou acesse a página no GitHub Pages.
